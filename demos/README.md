# Демо-расчёты для ссылки в блоге

## Как опубликовать

1. В калькуляторе соберите расчёт (откройте нужные вкладки: каналы / поддоны / экономика).
2. Нажмите **JSON** — скачается `daogreen-project-….json`.
3. Скопируйте файл сюда, например: `demos/klient-ivan-2026-05.json`  
   (только латиница, цифры, дефис; путь должен начинаться с `demos/`).
4. Закоммитьте и запушьте на GitHub Pages.
5. Ссылка для поста:

```
https://radagast1239.github.io/daogreen-calculator/calculator-110x55_12.html?share=demos/klient-ivan-2026-05.json
```

(подставьте свой файл и актуальный `?v=` из `index.html`, если меняли сборку)

## Пример

`demos/mangold-channels.json` — мангольд baby, 12 каналов, ВФ (как в `preview-config.js`).

Проверка локально: `start-server.bat` →  
`http://127.0.0.1:8080/calculator-110x55_12.html?share=demos/mangold-channels.json`

## Ограничения

- Режим **только просмотр**: нельзя менять поля; PDF и переключение вкладок — можно.
- Читатель видит **ваш файл**, не данные из своего браузера.
- Перед экспортом откройте вкладки, которые должны попасть в демо (см. обсуждение JSON в чате).
