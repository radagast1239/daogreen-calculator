Теплица

Значения для сорта —, в т.ч. шапка в мм. «Сохранить для сорта» пишет в браузер (localStorage) — остаётся после перезагрузки, пока не сбросите или не очистите данные сайта. «Применить» подставит в расчёт.

Вертикальная ферма

Сорт —. Поле «Шапка, мм» — вручную для сорта (не только из модели). «Сохранить для сорта» → localStorage навсегда в этом браузере. Галочки выше — что подставлять из справочника VF.

Подставлять из справочника VF:

Модель роста. Логистическая: M(t) = Mmax / (1 + e−k(t−t₅₀)), шапка d = c · √M
Температура. f(T) = exp(−(T−Topt)² / 2σ²), где Topt у каждого сорта свой: 20–22°C у прохладных, 23–24°C у батавий Rijk Zwaan (Афицион, Старфайтер и др.). Стрелкование ускоряется выше (Topt + 3°C).
Затенение. При перекрытии шапок Mmax снижается на 5–35% — нижние листья страдают от недостатка света. Включено автоматически, видно в рекомендациях.
Условия предполагаются идеальные: EC 1.4–1.8 мСм/см, pH 5.8–6.2, питание полностью сбалансировано. Параметры сортов: Aficion RZ (81-82), Afilion RZ, Starfighter RZ (81-85), Grazion RZ — данные Rijk Zwaan для индора/гидропоники, остальные сорта — Cornell CEA Lettuce Handbook (Brechner & Both), коммерческая практика NFT.
